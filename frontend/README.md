# Frontend (Website + Admin Console)

This Next.js app provides:
- public product website pages,
- a logged-in-style console shell,
- reusable UI primitives and mock data for product demos.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Mocked data

All current domain data is mock-backed via `lib/site-data.ts`.
Swap this file with API client hooks as backend contracts are finalized.
