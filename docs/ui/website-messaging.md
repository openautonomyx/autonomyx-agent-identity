# Website Messaging

## Core positioning
AutonomyX is an Agent Identity Plane for enterprises that need policy-governed, auditable agent operations.

## Messaging pillars
1. **Trust by default**: identity lifecycle and revocation for non-human actors.
2. **Policy in the request path**: OpenFGA + OPA enforcement model.
3. **Operational clarity**: audit, webhook events, health posture, and deployment guidance.
4. **Open and self-hostable**: production controls without SaaS lock-in.

## Audience
- Platform engineering
- Security engineering
- AI platform owners
- Governance and compliance stakeholders
