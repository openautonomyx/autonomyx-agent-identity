# Design System

## Tokens
- Base colors are defined in `frontend/tailwind.config.ts` (`bg`, `panel`, `border`, `accent`, status tones).
- Spacing follows Tailwind scale with card-first layout patterns.
- Typography: high-contrast headings with compact enterprise body text.

## Components
- `Badge`: status chip for lifecycle and dependency health.
- `PageHeader`: standardized page title/subtitle pattern.
- `PlaceholderPage`: controlled placeholder scaffolding for incomplete modules.
- `TopNav` and `SideNav`: marketing and console navigation shells.

## UX patterns
- Card-based information hierarchy for metrics and controls.
- Dense table layout for agent management workflows.
- Explicit placeholder language used where backend wiring is pending.
